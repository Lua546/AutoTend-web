Sobre el Proyecto

AutoTend surge como solución a un problema cotidiano usando tecnología accesible

El Problema

La ropa tendida al aire libre queda expuesta a lluvias inesperadas cuando no estamos en casa, causando que se moje y tengamos que lavarla nuevamente.

Objetivo

Desarrollar un sistema automático que detecte la lluvia y recoja el tendedero, protegiendo la ropa sin necesidad de intervención humana.

Beneficiarios

Personas ocupadas, familias, estudiantes, personas mayores y cualquiera que busque automatizar tareas del hogar ahorrando tiempo y esfuerzo.

Una solución inteligente

AutoTend combina hardware y software en un dispositivo compacto y eficiente. Utilizando la plataforma Arduino, sensores de lluvia y motores, hemos creado un sistema que:

  • Detecta la presencia de lluvia de forma automática
  • Activa un mecanismo de retracción del tendedero
  • Protege la ropa de condiciones climáticas adversas
  • Funciona sin supervisión constante
  • Es económico y fácil de implementar

Cómo Funciona

AutoTend opera de manera simple pero efectiva en tres pasos

1

Detección

El sensor de lluvia detecta las primeras gotas y envía una señal al Arduino

2

Procesamiento

El Arduino procesa la información y activa el motor de retracción

3

Protección

El tendedero se retrae automáticamente, poniendo la ropa a salvo

Código Arduino


// Definir los pines
const int sensorLluviaPin = A0;  // Pin conectado al sensor de lluvia
const int motorAdelantePin = 7;  // Pin conectado al motor para giro hacia adelante
const int motorReversaPin = 8;  // Pin conectado al motor para giro en reversa
const int motorVelocidadPin = 9; // Pin conectado para regular la velocidad

// Variable para rastrear el estado del sensor
bool estadoSensorAnterior = LOW; // Almacena el estado previo del sensor

void setup() {
  pinMode(sensorLluviaPin, INPUT);
  pinMode(motorAdelantePin, OUTPUT);
  pinMode(motorReversaPin, OUTPUT);
  pinMode(motorVelocidadPin, OUTPUT);

  Serial.begin(9600);

  // Asegurar que el motor esté apagado al inicio
  digitalWrite(motorAdelantePin, LOW);
  digitalWrite(motorReversaPin, LOW);
}

void loop() {
  int estadoSensorActual = digitalRead(sensorLluviaPin);
  Serial.println(digitalRead(sensorLluviaPin));
  if (estadoSensorActual == HIGH && estadoSensorAnterior == LOW) {
    // Comienza la lluvia: el sensor cambia de LOW a HIGH
    Serial.println("Empezó a llover");
    digitalWrite(motorAdelantePin, HIGH); // Activar motor hacia adelante
    analogWrite(motorVelocidadPin, 125);    // Velocidad al máximo
    delay(500);  // Mantener el motor encendido por 2 segundos
    digitalWrite(motorAdelantePin, LOW);  // Apagar el motor
  }
  else if (estadoSensorActual == LOW && estadoSensorAnterior == HIGH) {
    // Deja de llover: el sensor cambia de HIGH a LOW
    Serial.println("Dejó de llover");
    digitalWrite(motorReversaPin, HIGH); // Activar motor en reversa
    analogWrite(motorVelocidadPin, 125);    // Velocidad al máximo
    delay(500);  // Mantener el motor encendido por 2 segundos
    digitalWrite(motorReversaPin, LOW);  // Apagar el motor
  }

  // Actualizar el estado previo del sensor
  estadoSensorAnterior = estadoSensorActual;
}

Esquema de Conexiones

Componentes necesarios:
  • 1 x Arduino UNO o similar
  • 1 x Sensor de lluvia (YL-83)
  • 1 x Servo motor (SG90)
  • 1 x LED indicador
  • Cables de conexión
  • Fuente de alimentación (5V)

Resultados y Mejoras

Lo que hemos logrado y hacia dónde vamos

Logros del Proyecto

  • Detección confiable de lluvia con el sensor YL-83
  • Tiempo de respuesta menor a 5 segundos ante precipitaciones
  • Retracción completa del tendedero en menos de 10 segundos
  • Precisión del 95% en condiciones de prueba reales
  • Bajo consumo energético (promedio 150mA)

Desafíos Enfrentados

  • Calibración inicial del sensor para diferentes tipos de lluvia
  • Potencia limitada del motor para tendederos con mucha ropa
  • Durabilidad de componentes expuestos a la intemperie
  • Autonomía de la batería en implementaciones inalámbricas
  • Distinción entre lluvia real y humedad ambiental alta

Propuestas de Mejora

Conectividad IoT

Incorporar conectividad WiFi para monitoreo remoto y notificaciones al smartphone cuando se activa el sistema.

Energía Solar

Implementar un panel solar pequeño para hacer el sistema autosostenible y evitar cambios de batería.

Múltiples Sensores

Agregar sensores adicionales (humedad, temperatura) para predicciones más precisas y evitar falsos positivos.

Motor más Potente

Reemplazar el servo por un motor de mayor torque para soportar tendederos más cargados.

Interfaz LCD

Incorporar una pantalla LCD pequeña para visualizar el estado del sistema y parámetros ambientales.

Integración Domótica

Hacer el sistema compatible con plataformas de hogar inteligente como Home Assistant o Google Home.
